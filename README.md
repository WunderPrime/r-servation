# Système de Réservation pour Restaurant

Un projet PHP/MySQL complet pour gérer les réservations d’un restaurant avec **confirmation par email**.

---

## Fonctionnalités

- Formulaire de réservation sécurisé (CSRF + validation des champs)
- Stockage des réservations dans une base MySQL
- Confirmation automatique par email via **PHPMailer** et **Mailpit** (test local)
- Page de confirmation après soumission
- Interface simple et responsive en HTML

---

## Technologies utilisées

- PHP 8+
- MySQL / PDO
- PHPMailer pour l’envoi d’emails
- HTML
- Mailpit pour tester les emails localement
